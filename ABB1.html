Project Description:

The goal of this project is to become familiarised with the ABB IRB120 robot in order to use it in later projects.

The Kinematics movements are implemented in a MATLAB node. This node communicates with the ABB IRB120 robot using the Robot Operating System or ROS.

The scenario of the simulation is that a pile of rubble is falling down, and the robot is trying to repel them and keep its space clear by repeatedly pushing the blocks away.
